#  Contact Form with JavaScript Validation



This project is a simple Contact Form built using HTML, CSS, and JavaScript with client-side validation.

---

##  Objective

Build a contact form that validates:

- Name (non-empty)
- Email (valid format using Regex)
- Message (non-empty)

The form prevents submission if inputs are invalid and displays error messages dynamically.

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Regular Expressions (Regex)

---

##  Features

Styled responsive form  
Client-side validation  
Regex email validation  
Dynamic error messages  
`event.preventDefault()` to stop invalid submission  
Success message on valid submission  
Form reset after successful validation  

---

## Key Concepts Practiced

- Form Elements
- DOM Manipulation
- Event Handling
- Validation Logic
- Regex Pattern Matching
- User Feedback Messages
